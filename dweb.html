01 COMENCEMOS

En esta primera fase necesitamos conocerte: gustos, objetivos, clientes, tipo de industria, preferencias; así como toda la información que deseas mostrar en tu página web.

Se te pedirá que nos envíes la información estrucuturada, es decir, el contenido de cada una de las páginas/secciones que tendrá el sitio, en un archivo como: Word, PowerPoint o culaquier procesador de texto que mejor te acomode. *Todos los elementos deberán entregarse en versión digital.

Esta fase nos permitirá descubirir el estilo y estructura que mejor muestre tu contenido, siendo la base para conseguir el sitio web perfecto para tu negocio.

02 DISEÑO WEB A TU MEDIDA

Una vez obtenida toda la información necesaria, es momento de diseñar! Te propondremos colores, tipografías, imágenes que mejor representen a tu negocio.

Tenemos como objetivo encontrar el diseño adecuado y atractivo para ti; siempre proponiendo estructuras compatibles con buscadores e intuitivas que mejoren la navegación de tu audiencia y posibles consumidores.

Enviaremos un correo electrónico con las propuestas de diseño en JPG de cómo se verá tu sitio web, de las cuales podrás elegir la que mejor te caracterize. *Todas nuestras propuestas son únicas y creadas desde cero específicamente para tu negocio.

03 DESARROLLO Y PRUEBAS

Listo! El diseño web antes aprobado se vuelve un sitio completamente funcional; utilizamos lo último en lenguajes de programación como: HTML5, CSS2/Css3, React js, Java script, PHP para obtener compatibilidad con todos los navegadores modernos y dispositivos móviles; así es, su sitio web podrá verse en cualquier tamaño de pantalla. *Dependiendo de la complejidad y funcionalidad de cada proyecto se usarán los lenguajes adecuados.

Una vez finalizado, antes de lanzarlo, realizaremos las últimas pruebas; perfeccionando cada mínimo detalle para asegurar un óptimo funcionamiento. Terminando esta fase damos su página de alta en buscadores y está listo para estrenarse!
Si contrató nuestro servicio de administrador de contenidos, se le enviará por correo electrónico toda la documentación y videos detallados de como administrarlo.
